# ContentSearch
A tool for searching the content in doc and docx format

一个用来检索doc与docx文件的工具

##Support System

Linux

# Require

- 需要预先安装wv工具用来解析doc格式
- wv fro parse doc format

# Usage

show version `./cs --version`

# Install

1. make and make install

2. release/install.sh
